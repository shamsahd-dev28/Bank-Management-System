# 🏦 Bank Management System (Java)

## 📌 Overview

The Bank Management System is a simple Java application that allows users to perform basic banking operations. It is designed to demonstrate fundamental programming concepts and simulate real-world banking functionalities.

---

## 🚀 Features

* Create a new bank account
* Deposit money
* Withdraw money
* Check account balance
* View account details
* Transaction handling

---

## 🛠️ Technologies Used

* Java (Core Java)
* OOP Concepts (Classes, Objects, Inheritance, Encapsulation)
* File Handling (optional for data storage)
* Scanner class for user input

## 💡 Example Operations

* Create Account → Enter name, account number, initial balance
* Deposit → Add money to account
* Withdraw → Deduct money (if sufficient balance)
* Check Balance → Display current balance
